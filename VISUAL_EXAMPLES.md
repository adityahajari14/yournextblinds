## Code Example: How It Works

### Step 1: Product Data from API
```json
{
  "id": "123",
  "slug": "textured-vinyl-vertical",
  "title": "Textured Vinyl Vertical Blinds",
  "categories": [
    {
      "slug": "vertical-blinds",
      "name": "Vertical Blinds"
    }
  ]
}
```

### Step 2: Category Detection
```typescript
const categorySlug = apiProduct.categories[0]?.slug;
// Result: "vertical-blinds"
```

### Step 3: Feature Lookup
```typescript
const features = getCategoryCustomizations("vertical-blinds");
// Result:
{
  hasSize: true,
  hasHeadrail: true,
  hasOpenStyle: true,
  hasWandPosition: true,
  hasValance: true,
  hasBottomChain: true,
  hasBracketType: true,
  hasColour: true,
  hasRollerStyle: false,  // Hidden
  hasFabricType: false,   // Hidden
  hasBottomBar: false,    // Hidden
  hasLift: false,         // Hidden
}
```

### Step 4: Conditional Rendering
```tsx
{/* This WILL render */}
{product.features.hasHeadrail && (
  <HeadrailSelector ... />
)}

{/* This will NOT render */}
{product.features.hasRollerStyle && (
  <RollerStyleSelector ... />  // Hidden for vertical blinds
)}
```
